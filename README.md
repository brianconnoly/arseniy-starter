# arseniy-starter

Стартер проекта для генерации кода по спецификации

## Как пользоваться

1. Установить рекомендуемые расширения из .vscode/extensions.json

2. Включаем likec4-mcp в настройках плагина

3. Удалить всё из папки generated (пример генерации)

4. Описываем базовую архитектуру проекта в specs/workspace.c4 (можно с помощью ии-агента)

5. Просим сгенерировать проект по спецификации

6. Далее можем менять структуру и просить агент отразить это в коде, либо давать прямые задания на доработку продукта
